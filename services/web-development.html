// Case Study

Real Results, Real Impact

Website Rebuild & Cloud Migration for a UK Charity

🌐 Non-Profit & Charity Sector
Challenge

A UK-based cultural charity with 5,000+ members was running a WordPress site on a £15/month shared hosting plan. The site was painfully slow (8+ second load times), frequently crashed during event registration periods (losing 30-40% of potential registrations), had an outdated design that did not reflect the organisation's prestige, and was running WordPress 5.2 with 23 outdated plugins — several with known security vulnerabilities. The charity needed a modern, fast, reliable website capable of handling event registrations, membership management, and donation processing.

Solution

TotalCloudAI redesigned the entire website with a modern, accessible design (WCAG 2.1 AA compliant) using a custom WordPress theme with Tailwind CSS. We migrated the site to Azure App Service for Linux with Azure CDN, Azure Database for MySQL, Redis Object Cache, and automated daily backups to geo-redundant storage. CI/CD was configured through GitHub Actions with a staging environment for content preview. The event registration system was rebuilt using WooCommerce with Stripe integration, capable of handling 500+ concurrent registrations. We implemented Core Web Vitals optimisation achieving LCP under 1.5 seconds, configured Cloudflare WAF for security, and set up automated WordPress core and plugin updates with staging-first testing.
